          Advanced features below are experimental and may change. Use with caution in production environments.
          
            
          
            
          
                How it works: When trackers send requests, we intercept and replace their data with synthetic information using detection patterns (schemas). 
                Each schema targets specific tracking services and generates consistent fake data per-origin to poison their datasets and resist behavioral analysis.
            
          
            
            
          
                
          Built-in Detection Patterns:
          
                
            
                    
          • YouTube QoE: Replaces video quality metrics with synthetic bandwidth/buffer data
            • 
                    
          • Facebook Pixel: Generates fake user IDs and event data for Meta tracking
            • 
                    
          • TikTok Pixel: Injects synthetic device/location data for ByteDance analytics
            • 
                    
          • Generic Analytics: Handles Google Analytics, Mixpanel, Segment, Hotjar, FullStory
            • 
                
          
                
          
                    💡 User-configurable: You can modify detection patterns by editing schemas/adultPersona.js 
                    to add new services, change synthetic data, or adjust targeting rules.
